## Tech Stack

**Client:** React, TailwindCSS

**Server:** NodeJS, ExpressJS

**Database:** MongoDB, Mongoose

## Other Features

- Profile
- Dark Theme

## Setting Up

Install dependencies:

```bash
  npm install
```

Finally,

```bash
  npm start
```
